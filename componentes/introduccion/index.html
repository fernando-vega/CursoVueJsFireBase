Componentes | Que son?

El nombre que le dé a un componente puede depender de dónde quiera usarlo. Cuando se usa un componente directamente en el DOM (a diferencia de en una plantilla de cadena o componente de un solo archivo), recomendamos encarecidamente seguir las reglas W3C para nombres de etiqueta personalizados (todo en minúsculas, debe contener un guión). Esto le ayuda a evitar conflictos con elementos HTML actuales y futuros.
